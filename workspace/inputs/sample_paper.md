# Sample Paper

This is a placeholder paper text for the MVP.

## Problem
Researchers need a lightweight agent that can read papers and help generate structured writing artifacts.

## Method
The system uses a sequence of skills to ingest text, summarize content, extract claims, build an outline, and draft a write-up.

## Experiments
The current MVP focuses on workflow validation rather than benchmark performance.

## Limitations
Only txt and markdown inputs are supported in the first version.
