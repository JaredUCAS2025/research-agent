# Memory: 用户偏好与行为记忆

> 本文件记录用户的长期偏好和行为规则。每次任务执行前，系统会读取此文件以保持行为一致性。
> 用户可通过对话指令添加、修改或删除记忆条目。

---

## 输出偏好

- 默认使用中文输出。
- 术语保留英文原文并加括号注释。
- 偏好结构化输出：使用标题层级、列表、表格，而非大段文字。

## 分析偏好

- 单篇论文分析时，优先关注：方法创新点、实验设置、与现有工作的区别。
- 多篇论文对比时，重点关注：方法差异、评测指标一致性、结论冲突。
- 综述生成时，按"问题定义 → 方法演进 → 实验对比 → 开放问题"的结构组织。

## 交互偏好

- 回答尽量简洁，避免重复论文原文中已明确的内容。
- 不确定的内容直接说"论文未提及"或"需要进一步确认"。
- 追问时优先利用已有 artifacts，减少重复分析。
